WEBSERVICES Using LARAVEL 10 MySQL

# Backend 
### Installing 
```
composer install
```


### Setup Environment Variable
```
DATABASE_NAME = 'database'
DATABASE_USERNAME = 'root'
DATABASE_PASSWORD = ''
```
### Running Migrate - Database
```
php artisan migrate
```
### Running Webservices - Restful API
```
php artisan serve
```
